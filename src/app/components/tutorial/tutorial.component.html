Tutorial

Open Enzyme Database offers curated data on enzyme kinetics and experimental parameters, along with advanced prediction and recommendation algorithms, in a unified web-based interface. Below, you’ll find descriptions and examples of each of these core workflows.
Database Search
The database search functionality enables users to discover curated data on enzyme kinetics parameters with experimental support. After inputting an entity (compound, organism, Uniprot ID, EC Number, pH, or temperature), users receive an interactive list of relevant data, which they can filter, sort, or drill down into to explore detailed information for compounds, EC Numbers, and Uniprot IDs. Outlinks to reference databases such as PubMed, Uniprot, and BRENDA, as well as export to CSV, are also supported.
Multiple entity types are supported for database search. Here, the compound Ethanol (SMILES: COO) has been input.
After clicking ‘Search Open Enzyme Database,’ relevant experimental results are displayed in an ‘Experimental Data’ table. Users can expand individual table rows to access additional information such as reference data outlinks and reaction schemes..
Individual entity pages - which can be accessed by clicking any of the blue internal links - provide detailed information and further filterable data for Compounds, EC Numbers, and Uniprot IDs. Here, the Compound entity page for Ethanol is displayed.
Enzyme Property Prediction
The enzyme property prediction functionality enables users to predict kcat, KM, kcat/KM (catalytic efficiency), or Ki (inhibitory constant) values for enzyme-substrate pairs using 3 novel ML algorithms: DLKcat, UniKP, and CatPred:
  • DLKcat employs a deep learning approach that combines a graph neural network for substrate representation with a convolutional neural network for protein representation to predict the kcat kinetic parameter from enzyme-substrate pairs.
  • UniKP is a unified framework for predicting enzyme kinetic parameters (including kcat, KM and catalytic efficiency kcat/KM) from amino acid sequences and substrate pairs using pre-trained language models.
  • CatPred is a deep learning model for predicting in vitro enzyme kinetic parameters (including kcat, KM, and inhibitory constant Ki) by leveraging pre-trained protein language model and Directed Message Passing Neural Networks (D-MPNN).
A single enzyme-substrate pair input allows users to run all three prediction models at once. Here, the enzyme sequence WP_063460136 (FASTA) and substrate Ethanol (SMILES: COO) have been input for analysis.
After a short processing time, prediction results are returned for all three models. The results summary page provides a high-level overview of each enzymatic property and its predicted value for each model.
Clicking ‘View full results’, users can navigate into detailed prediction results pages for each model. Here, the predicted values for each parameter are contextualized with respect to the corresponding experimental values reported in Open Enzyme Database.
Enzyme Recommendation
The enzyme recommendation functionality enables users to discover enzyme recommendations for a specific substrate using 3 structure search algorithms: MCS, Tanimoto Similarity, and Fragment Algorithm:
  • MCS identifies the largest shared substructure between two molecules, highlighting core structural similarities that may influence enzyme compatibility, where a higher ‘MCS Score’ indicates greater similarity.
  • Tanimoto Similarity calculates similarity based on molecular fingerprints, quantifying how much two molecules share in common relative to their overall structure, where a higher ‘Tanimoto Score’ indicates greater similarity.
  • Fragment Algorithm compares molecules by breaking them into smaller chemical fragments and assessing overlap, capturing similarities in localized functional groups or motifs, where larger fragment sizes indicate greater similarity.
A single substrate input allows users to run all three structure search algorithms at once. Here, the substrate Ethanol (SMILES: COO) has been input for analysis.
After a short processing time, recommendation results are returned for all three models. The results summary page provides a high-level overview of the top substrate matches and associated scores for each model.
Clicking ‘View full results’, users can navigate into detailed recommendation results pages for each model. Here, relevant experimental results are organized by compound, and expandable table rows provide access to compound and enzyme structure visualizations.
